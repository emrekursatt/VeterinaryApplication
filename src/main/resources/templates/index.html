Veterinary Application

Sistem üzerinde iki tür kullanıcı yetkisi vardır. / There are two types of user authority on the system.

User Kullanıcı sistemde neler yapabilir? / User what can the user do in system?

•Sisteme üye olabilir ve sonrasında giriş yapabilir. • Sisteme hayvan sahibi ve hayvan ekleme, güncelleme gibi işlemleri yapabilir. • Hayvan adı veya hayvan sahibi adı ile arama yapabilir ve kayıtları görüntüleyebilir. • Sisteme kayıtlı hayvan sahibi sayısını ve hayvan sayısını görebilir.

•He/she can become a member of the system and then log in. • He/she can carry out operations such as owner and adding animals to the system, updating them. • He/she can search by animal name or owner name and view records. • He/she can see the number of animal owners and the number of animals registered in the system

Admin kullanıcı sistemde neler yapabilir ? / What can the admin user do in the system?

User kullanıcının tüm yetkilerine sahiptir. Ancak sistemde silme rolü sadece admin kullanıcısına verilmiştir.

Admin user has all privileges of the user. However, the delete role in the system is given only to the admin user.

Kullanılan teknolojiler: Spring Boot + Spring MVC + Spring Security + JPA/Hibernate + MySQL + Maven + HTML + Thymeleaf

Used technologies: Spring Boot + Spring MVC + Spring Security + JPA/Hibernate + MySQL + Maven + HTML + Thymeleaf
